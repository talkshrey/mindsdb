MindsDB Cloud is a service hosted by MindsDB. It contains all of the latest
  updates and provides a handy SQL editor so you can run your queries right
  away.

You can sign up for
[a free account at MindsDB Cloud here](https://cloud.mindsdb.com/register).
Follow the steps below that guide you through the sign-up process.

After you sign up for a free MindsDB Cloud account, we'll send you a
confirmation email. Click on the _Verify Email_ button to validate your account.
